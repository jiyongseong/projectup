# Azure 주간 업데이트 요약 (2023-10-05)

## ⚙️ Storage  
Azure Storage는 데이터 보호, 복원력 및 이동성을 개선하기 위해 여러 업데이트를 발표했습니다. **Azure NetApp Files 고급 랜섬웨어 방지 기능**이 미리 보기로 제공되었으며, 이를 통해 클라우드 환경에서 랜섬웨어를 탐지하고 복구하는 기능이 강화되었습니다. 또한 **Azure NetApp Files 교차 영역-지역 간 복제**가 정식 지원으로 출시되어 데이터 복원력과 비즈니스 연속성에 기여합니다. 마지막으로 **Azure Storage Mover**의 신규 Blob-to-Blob 이동 기능은 클라우드 내 데이터 전송을 쉽게 하는 데 도구를 제공합니다.

### [Azure NetApp Files 고급 랜섬웨어 보호(미리 보기)](https://azure.microsoft.com/updates?id=536699)
Azure NetApp Files의 랜섬웨어 보호 기능은 클라우드 환경에서 랜섬웨어 위협을 탐지, 대응 및 복구를 가능하게 합니다.

### [Azure NetApp Files 교차 영역-지역 복제(GA)](https://azure.microsoft.com/updates?id=537106)
교차 영역 및 지역 간 복제를 통해 재해 복구와 비즈니스 연속성을 지원하는 기능이 정식 출시되었습니다.

### [Azure Storage Mover의 Blob-to-Blob 이동 기능(미리 보기)](https://azure.microsoft.com/updates?id=542813)
Azure Storage Mover는 Blob 간 데이터를 간단하고 효율적으로 이동할 수 있는 기능을 제공합니다.

---

## 🗂️ Databases  
데이터베이스 관련 업데이트는 성능 향상을 위한 새로운 도구 제공에 초점이 맞춰졌습니다. **Azure Cosmos DB용 SRE 에이전트**가 미리 보기로 출시되어 데이터베이스 상태 점검과 문제 해결을 지원합니다. 또한 **Azure SQL 업데이트**는 서버리스 워크로드를 원활히 재개할 수 있도록 데이터베이스 활동 로그를 분석하는 기능을 포함합니다.

### [Azure Cosmos DB 위한 SRE 에이전트(미리 보기)](https://azure.microsoft.com/updates?id=541813)
SRE 에이전트를 통해 데이터베이스 성능 문제를 진단하고 해결할 수 있습니다.

### [Azure SQL 업데이트(정식 지원)](https://azure.microsoft.com/updates?id=541818)
Azure SQL에는 서버리스 워크로드 재개를 지원하는 새로운 기능이 포함됩니다.

---

## 🛠️ Compute  
Azure Compute는 개발자를 위한 새로운 기능을 확장하며, 특히 **Azure Functions**의 Service Bus SDK와 연결된 Node.js 타입 바인딩에 업데이트를 추가하여 고급 메시징 기능을 강화했습니다. 이를 통해 더욱 효율적인 함수 애플리케이션 개발이 가능해졌습니다.

### [Azure Functions Service Bus SDK 타입 바인딩(미리 보기)](https://azure.microsoft.com/updates?id=541427)
Azure Functions는 Service Bus와 연결된 새로운 SDK 타입 바인딩을 제공하여 개발 환경을 간소화합니다.

---

## 🚀 Migration  
Azure Migration은 기존 데이터를 클라우드 간 이동시키는 과정을 간소화하는 데 중점을 두었습니다. **Azure Storage Mover**가 **Blob-to-Blob 이동** 기능을 미리 보기로 발표하며, 이는 사용자가 대규모 데이터 이전 과정을 쉽고 안전하게 수행할 수 있는 도구를 제공합니다.

### [Azure Storage Mover, Blob-to-Blob 이동(미리 보기)](https://azure.microsoft.com/updates?id=542813)
클라우드간 데이터를 안전하고 효율적으로 이동할 수 있는 기능을 제공합니다.

---

## 🌍 Hybrid + Multicloud  
하이브리드 및 멀티 클라우드 환경을 지원하기 위해 **Azure SQL Database**가 자동화된 워크로드 관리를 포함한 새로운 기능을 추가했습니다. 이러한 업데이트는 데이터 작업을 최적화하고 효율성을 높이는 데 기여합니다.

### [Azure SQL 데이터베이스 업데이트(정식 지원)](https://azure.microsoft.com/updates?id=541818)
하이브리드 클라우드 환경에서 데이터 작업을 간소화하고 최적화할 수 있는 업데이트가 포함되었습니다.
