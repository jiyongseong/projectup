# Azure 주간 업데이트 요약 - 2023년 10월 12일

## 🗄️ 스토리지 업데이트
지난 주 스토리지 관련 업데이트는 Azure NetApp Files와 Azure Storage Mover에 초점이 맞춰졌습니다. Azure NetApp Files는 랜섬웨어 방지를 위한 고급 보호 기능과 크로스-존-지역 복제 기능을 통해 더 안전하고 강력한 데이터 관리 옵션을 제공합니다. 또한, Azure Storage Mover는 Blob-to-Blob 마이그레이션을 간소화하여 데이터가 보관되는 위치 간에 보다 쉽게 이동할 수 있도록 지원합니다. 이러한 개선은 클라우드 스토리지에서의 보안, 데이터 이동 효율성 및 관리 용이성을 눈에 띄게 향상시킵니다.

### [Azure NetApp Files 고급 랜섬웨어 보호 (미리 보기)](https://azure.microsoft.com/updates?id=536699)
Azure NetApp Files는 클라우드 볼륨에서 랜섬웨어 위협을 사전에 탐지하고 신속히 대응 및 복구할 수 있는 능력을 제공합니다. 위협을 탐지한 즉시 스냅샷을 생성하여 데이터를 복원할 수 있습니다.

### [Azure NetApp Files 크로스-존-지역 복제 (정식 지원)](https://azure.microsoft.com/updates?id=537106)
이 기능은 재해 복구 및 비즈니스 연속성을 지원하며 볼륨을 지역과 가용성 영역 간에 복제할 수 있는 기능을 제공합니다.

### [Blob-to-Blob 마이그레이션 간소화](https://azure.microsoft.com/updates?id=542813)
Azure Storage Mover를 통해 Blob 컨테이너 간의 데이터 이동이 자동화되고 간편해졌습니다. 사용자는 포털에서 작업을 설정하고 실시간으로 작업 상태를 확인할 수 있습니다.

## 🛠️ 데이터베이스 업데이트
지난 주 데이터베이스 관련 업데이트는 Azure Cosmos DB와 Azure SQL에서 새롭게 도입된 기능들이 주목을 받았습니다. Azure Cosmos DB는 Site Reliability Engineering(SRE) Agent로 데이터베이스 워크로드 관리를 간소화하고 더욱 상세하게 모니터링할 수 있도록 합니다. 또한, Azure SQL은 서버리스 워크로드 관리를 개선할 수 있는 기능 강화로 사용자 편의성을 증가시켰습니다.

### [Azure Cosmos DB SRE Agent 활용 (미리 보기)](https://azure.microsoft.com/updates?id=541813)
애플리케이션 문제 분석을 간소화하며, SRE 에이전트를 통해 데이터베이스 성능과 보안을 강화하고 비용을 최적화할 수 있습니다.

### [Azure SQL 업데이트 (정식 지원)](https://azure.microsoft.com/updates?id=541818)
Azure SQL은 액세스 패턴 최적화와 문제 해결을 위한 Activity Log 기능 등 새로운 업데이트가 적용되었습니다.

### [Activity Log를 통한 분석 최적화](https://aka.ms/AAxxog6)
서버리스 워크로드에서 발생하는 문제의 원인을 분석하고 최적화된 액세스 패턴 설계를 지원하는 도구입니다.

## 💻 컴퓨팅 및 컨테이너 업데이트
이번에는 Azure 컴퓨팅 부문에서 Azure Functions의 확장된 SDK 타입 바인딩 지원이 중점적인 업데이트로 다뤄졌습니다. Service Bus SDK 타입 바인딩이 추가되면서 JavaScript 및 TypeScript 기반 함수 앱 개발자들에게 더 나은 개발 환경을 제공합니다.

### [Service Bus SDK 타입 바인딩 도입 (미리 보기)](https://azure.microsoft.com/updates?id=541427)
Service Bus에 대한 SDK 바인딩 지원이 JavaScript 및 TypeScript에서도 제공되기 시작하여 개발자 워크플로가 더욱 간소화되었습니다.

### [SDK 타입 바인딩 문서](https://aka.ms/sdktypesnodesb)
SDK 타입 바인딩의 작동 원리와 활용 방법을 설명하는 공식 문서로 가이드를 제공합니다.

### [SDK 타입 바인딩 샘플](https://aka.ms/sdktypesnodesbsamples)
개발자들이 활용할 수 있는 샘플 코드 및 상세한 예제를 제공합니다.
