# Azure 주간 업데이트 요약 - 2025년 12월 18일

## 💾 스토리지
Azure의 스토리지와 관련된 이번 주 주요 업데이트는 데이터 관리 및 보안을 강화시키는 데 중점을 둔 다양한 기능들이 포함되어 있습니다. 특히 Azure NetApp Files의 고급 랜섬웨어 보호 기능과 교차 영역 및 지역 복제의 정식 지원은 데이터 보호 및 복구에 큰 도움이 됩니다. 이외에도 Azure Storage Mover를 사용한 Blob-to-Blob 데이터 이동 기능은 클라우드 데이터 이전을 보다 간소화된 방식으로 처리할 수 있도록 해줍니다.

### [Azure NetApp Files 고급 랜섬웨어 보호 기능 (미리 보기)](https://azure.microsoft.com/updates?id=536699)
Azure NetApp Files는 클라우드 볼륨에서 랜섬웨어 위협을 탐지하고 복구할 수 있는 고급 랜섬웨어 보호 기능을 제공합니다. 이는 파일 활동을 모니터링하고 위협이 탐지되면 스냅샷을 생성해 데이터를 복구할 수 있도록 도와줍니다.

### [Azure NetApp Files 교차 영역 및 지역 복제 (정식 지원)](https://azure.microsoft.com/updates?id=537106)
교차 영역 및 지역 복제는 데이터 복구 및 지속성을 강화하는 기능으로, 이제 정식 지원됩니다. 이를 통해 사용자는 여러 영역 및 지역 간 데이터 복제를 설정하여 재해 복구를 더욱 효율적으로 관리할 수 있습니다.

### [Azure Storage Mover를 사용한 Blob-to-Blob 데이터 이동 (미리 보기)](https://azure.microsoft.com/updates?id=542813)
Azure Storage Mover를 사용하면 클라우드에서 Blob 컨테이너 간 데이터 이동이 더욱 간단하고 안정적으로 가능해집니다. 이는 대량의 데이터도 병렬 전송 방식을 통해 신속히 이전할 수 있도록 지원합니다.

## 🌐 데이터베이스
이번 주 데이터베이스 카테고리의 업데이트는 Azure의 데이터 관리 및 최적화 기능을 더욱 확장시키는 데 중점을 두고 있습니다. 특히 새롭게 소개된 Azure Cosmos DB의 사이트 안정성 엔지니어링(SRE) 에이전트는 데이터베이스 워크로드의 가시성을 크게 향상시키며, Azure SQL의 업데이트는 데이터 성능을 최적화하는 데 도움을 줍니다.

### [Azure Cosmos DB에서 Azure SRE 에이전트 사용 (미리 보기)](https://azure.microsoft.com/updates?id=541813)
Azure SRE 에이전트는 클라이언트와 서버 측 진단 정보를 통합하여 데이터베이스 상태와 성능을 보다 명확히 파악할 수 있도록 도와줍니다. 이를 통해 문제를 신속히 탐지하고 해결할 수 있습니다.

### [Azure SQL 업데이트 - 2025년 12월 (정식 지원)](https://azure.microsoft.com/updates?id=541818)
Azure SQL의 최신 업데이트는 서버리스 워크로드를 자동으로 재개하는 원인을 활동 로그를 통해 파악할 수 있습니다. 이를 통해 더욱 데이터를 최적화하고 접근 패턴을 개선할 수 있습니다.

Azure Functions의 새로운 바인딩 기능은 JavaScript 및 TypeScript 애플리케이션에서 Service Bus 메시지를 더 쉽게 처리하고 관리할 수 있도록 지원합니다.

## 📦 마이그레이션
클라우드 간 데이터 이동을 간소화하고 최적화하는 데 초점을 맞춘 Azure의 마이그레이션 업데이트는 조직이 데이터 이전 작업을 보다 효과적으로 수행할 수 있도록 만들어줍니다.

### [Azure Storage Mover를 사용한 Blob-to-Blob 데이터 이동 (미리 보기)](https://azure.microsoft.com/updates?id=542813)
Azure Storage Mover는 효율적이고 대규모로 확장 가능한 Blob-to-Blob 데이터 마이그레이션 기능을 제공합니다. 이를 통해 기업은 복잡한 수작업 데이터 이전 과정을 제거하고 신뢰성을 높일 수 있습니다.

### [Azure Functions에서 Service Bus SDK 바인딩 제공 (미리 보기)](https://azure.microsoft.com/updates?id=541427)
