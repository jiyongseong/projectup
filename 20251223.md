# Azure 주간 업데이트 요약 - 2023년 11월 17일

## 🗂 스토리지 (Storage)
Azure 스토리지 서비스는 데이터 이동성과 보안 강화를 중심으로 개선되었습니다. Azure NetApp Files는 랜섬웨어 위협을 선제적으로 탐지하는 고급 보호 기능이 추가되었으며, 크로스 존-리전 복제(CZRR)도 정식 지원 상태에 도달했습니다. 또한, Azure Storage Mover는 Blob 간 마이그레이션 기능을 간소화하여 대규모 데이터 이동 작업의 효율성을 극대화했습니다.

### [Azure NetApp Files 고급 랜섬웨어 보호 (미리 보기)](https://azure.microsoft.com/updates?id=536699)
Azure NetApp Files는 랜섬웨어 위협을 사전에 탐지하고 복구할 수 있도록 설계된 고급 보호 기능을 공개 미리 보기로 제공합니다. 주요 특성으로 파일 확장자 분석, 데이터 활동 모니터링, 및 즉각적인 스냅샷 등이 포함됩니다.

### [Azure NetApp Files 크로스 존-리전 복제 기능 정식 지원](https://azure.microsoft.com/updates?id=537106)
이제 Azure NetApp Files에서 크로스 존-리전 복제를 통해 가용성 존 간 데이터를 복제할 수 있으며, 재해 복구와 비즈니스 연속성을 결합할 수 있는 기능을 제공합니다.

### [Azure Storage Mover를 이용한 Blob-to-Blob 마이그레이션 (미리 보기)](https://azure.microsoft.com/updates?id=542813)
데이터 컨테이너 간의 마이그레이션이 보다 직관적이고 안전하게 개선되었습니다. Azure Storage Mover는 전용 에이전트 없이도 쉽게 설정할 수 있으며, 고속 병렬 전송 기능으로 대규모 데이터를 효율적으로 전송할 수 있습니다.

## 📊 데이터베이스 (Databases)
데이터베이스 관리 및 성능 최적화를 목표로 Azure SQL과 Azure Cosmos DB에서 다양한 개선 사항이 발생했습니다. Azure SQL은 서버 없는 워크로드를 최적화하는 로그 기능을 추가했으며, Azure Cosmos DB는 새롭게 도입된 SRE 에이전트를 통해 작업을 간소화할 수 있도록 했습니다.

### [Azure Cosmos DB SRE 에이전트 기능 공개 (미리 보기)](https://azure.microsoft.com/updates?id=541813)
Azure Cosmos DB는 시스템 상태와 워크로드의 효율성을 분석할 수 있도록 SRE 에이전트를 도입했습니다. 이 기능은 서버 진단 및 클라이언트 텔레메트리를 통합하여 심도 있는 분석과 최적화를 지원합니다.

### [Azure SQL 서버 없이 자동 구동 원인 분석 기능 정식 지원](https://azure.microsoft.com/updates?id=541818)
Azure SQL 데이터베이스는 자동 Activity Log 생성으로 서버 없는 워크로드의 구동 원인을 파악 및 최적화할 수 있습니다. 이를 통해 더 나은 접근 패턴 구성과 문제 해결이 가능합니다.

## 🚀 컴퓨트 + 컨테이너 (Compute + Containers)
Azure는 프로그래밍 언어 및 개발 환경을 지원하며 통합과 간소화를 지속적으로 강화하고 있습니다. 특히, Node.js와 TypeScript 애플리케이션 개발 시 Azure Functions의 효율성이 증대되었습니다.

### [Service Bus SDK 타입 바인딩 (미리 보기)](https://azure.microsoft.com/updates?id=541427)
Node.js에서 Azure Functions를 사용할 때 Service Bus SDK 타입을 지원하여 메시징 기능이 향상되었으며, 이는 JavaScript와 TypeScript 개발자들에게 특별히 유용합니다.

## 🌐 네트워킹 (Networking)
네트워킹 부문에서는 보안과 규정 준수에 중점을 둔 업데이트가 등장했습니다. 특히, Application Gateway V2 SKU는 이제 FIPS 모드를 지원하여 암호화와 보안을 강화했습니다.

### [Application Gateway V2 SKU FIPS 준수 기능 정식 지원](https://azure.microsoft.com/updates?id=536712)
미국 정부의 보안 표준인 FIPS 140-2 기준을 만족하는 애플리케이션 게이트웨이 구성 옵션이 추가되었습니다. 이는 FedRAMP 요구 사항을 포함한 여러 규정 준수 사양을 충족하도록 돕습니다.

## 🌏 사물인터넷 (IoT)
사물인터넷 관련 Azure Sphere OS가 새로운 평가 버전을 발표했습니다. Azure Sphere 고객들은 입주가 연장된 평가 기간 중 새 빌드를 테스트할 수 있습니다.

### [Azure Sphere OS 25.12 평가 버전 출시](https://azure.microsoft.com/updates?id=541784)
Azure Sphere OS의 25.12 버전이 평가판으로 제공됩니다. 해당 업데이트는 내부 빌드 시스템에서 중요한 변경 사항이 포함되며, 고객은 설정 평가 시간을 늘릴 수 있습니다.
