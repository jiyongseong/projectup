# Azure 주간 업데이트 요약 - 2026-02-16

## 🚀 새롭게 출시된 기능 (정식 지원)

Azure는 정식 지원 상태로 여러 중요한 업데이트를 발표하여 더욱 안정적이고 기능 강화된 서비스를 제공합니다. 이를 통해 기업의 데이터 처리 및 운영 능력이 획기적으로 향상될 수 있습니다. 지난 주에는 Azure Databricks의 Supervisor Agent와 AKS Kubernetes의 새로운 버전에 대한 업데이트가 발표되었습니다. 이 두 가지는 클라우드 운영과 머신러닝 워크플로우에서 보다 높은 효율성과 유연성을 제공하는 데 초점을 맞추고 있습니다. 

### [Azure Databricks Supervisor Agent](https://azure.microsoft.com/updates?id=557081)
Azure Databricks에서 제공되는 Supervisor Agent는 조직의 데이터 워크플로우를 더욱 스마트하게 관리하며, 분열된 단일 기능 에이전트를 통합해주는 역할을 합니다.

### [AKS Kubernetes 버전 1.34 정식 지원](https://azure.microsoft.com/updates?id=548114)
AKS는 이제 Kubernetes 버전 1.34를 정식으로 지원하며, 이를 통해 더 나은 클러스터 관리와 업데이트 계획이 가능해졌습니다.

### [Visual Studio Code에서 SQL 프로젝트 게시](https://learn.microsoft.com/en-us/sql/tools/visual-studio-code-extensions/sql-database-projects/publish-database-project?view=sql-server-ver17)
Visual Studio Code를 통해 신속하고 효율적으로 SQL 프로젝트를 배포할 수 있는 기능이 정식 출시되었습니다.

## 🔍 미리 보기 상태 (In Preview)

Azure는 미리 보기 상태로 여러 새로운 기능들을 제공하며, 사용자들이 이를 테스트하고 피드백을 공유할 수 있도록 합니다. 이번 주에는 컨테이너용 애플리케이션 게이트웨이 및 Azure Front Door Premium의 새로운 지역 지원과 Azure SQL의 중간 업데이트가 주요 미리 보기 상태의 기능으로 발표되었습니다.

### [컨테이너용 애플리케이션 게이트웨이](https://azure.microsoft.com/updates?id=555603)
AKS에 대한 관리형 애드온으로 컨테이너용 애플리케이션 게이트웨이를 간편하게 배포하고 관리할 수 있습니다.

### [Azure Front Door Premium UAE North 지역 지원](https://azure.microsoft.com/updates?id=556282)
UAE North 지역에서 Azure Private Link를 사용하는 Azure Front Door Premium을 설정할 수 있게 되었습니다.

### [Azure SQL 중간 업데이트 2026년 2월](https://azure.microsoft.com/updates?id=550159)
Azure SQL Database의 장애조치 그룹에서 새로 추가된 기능을 포함하는 업데이트가 진행됩니다.

## 🌐 네트워킹 및 보안

Azure는 네트워킹과 보안 기능 강화를 통해 클라우드 환경의 연결성 및 보안성을 높이고 있습니다. 이번 주의 주요 업데이트에는 컨테이너용 게이트웨이, SQL 장애 복구 그룹 및 Azure Private Link의 새로운 지역 지원이 포함됩니다.

### [컨테이너용 애플리케이션 게이트웨이 개요](https://aka.ms/agc/addon)
Azure 네트워킹 솔루션에 통합된 관리형 설치 및 업데이트 기능으로 관리가 편리해졌습니다.

### [Azure SQL Database 장애조치 그룹](https://learn.microsoft.com/en-us/sql/database/failover-group-sql-db?view=azuresql#multiple-secondaries)
Azure SQL의 장애 복구 기능이 확장되었으며, 최대 4개의 복제본을 구성할 수 있어 다지역 재난 복구 설계가 쉬워졌습니다.

### [Azure Private Link 리전 가용성](https://learn.microsoft.com/en-us/azure/frontdoor/private-link#region-availability)
Azure Front Door와 함께 사용하는 Private Link의 리전 가용성이 확대되었습니다.

## 🤖 AI 및 머신러닝

AI 및 머신러닝 서비스는 Azure 클라우드에서 데이터 분석 및 예측을 강화하며, 이를 통해 운영 효율성을 극대화할 수 있습니다. 이번 주에는 Databricks Supervisor Agent와 관련된 중요한 업데이트가 발표되었습니다.

### [Azure Databricks Supervisor Agent 소개](https://learn.microsoft.com/en-us/azure/databricks/generative-ai/agent-bricks/multi-agent-supervisor)
Supervisor Agent는 다중 에이전트 시스템을 통합하여 데이터 워크플로우의 성능을 극대화합니다.

### [Databricks 내 에이전틱 자동화](https://azure.microsoft.com/updates?id=557081)
Databricks 플랫폼을 기반으로 보다 효율적인 에이전트 관리 및 자동화가 가능합니다.

### [Kubernetes v1.34 업데이트](https://kubernetes.io/blog/2025/08/27/kubernetes-v1-34-release/)
새로운 Kubernetes 업데이트에는 워크플로우 지정 및 데이터 관리 효율성을 높이는 여러 개선 사항이 포함되어 있습니다.

## 📦 컨테이너 및 하이브리드 클라우드

컨테이너 및 하이브리드 클라우드는 멀티 클라우드 환경에서도 효과적인 관리와 확장성을 제공합니다. 이번 업데이트에서는 AKS의 Kubernetes 버전 지원과 관련된 사항들이 포함됩니다.

### [AKS Kubernetes 버전 지원](https://azure.microsoft.com/updates?id=548114)
새로운 Kubernetes 버전 1.34를 통해 컨테이너화된 애플리케이션의 클러스터 조작이 한층 강화되었습니다.

### [관리형 게이트웨이 API 크래들](https://aka.ms/agc/addon)
새로운 관리형 Gateway API 애드온은 AKS와 통합될 수 있어 관리의 편리성을 제공합니다.

### [Visual Studio 코드 연결 마이그레이션](https://github.com/microsoft/vscode-mssql/releases/tag/v1.39.0)
Azure Data Studio에서 Visual Studio Code로 워크플로우를 안전하게 이전하는 기능이 새로 제공됩니다.
