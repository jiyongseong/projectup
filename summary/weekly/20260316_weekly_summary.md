# Azure 주간 업데이트 요약 - 2026년 03월 16일

## 💡 Microsoft Foundry

Microsoft Foundry는 GPT-5.4 정식 지원과 Fireworks AI 미리보기 출시 등 인공지능과 AI 기반 모델 배포 분야에서 주목할 만한 업데이트를 발표했습니다. 이들은 워크플로우 효율성을 극대화하고, AI 모델을 확장성과 신뢰성을 기반으로 관리할 수 있게 돕습니다.

### [GPT-5.4 정식 지원](https://techcommunity.microsoft.com/t5/microsoft-foundry-blog/introducing-gpt-5-4-in-microsoft-foundry/ba-p/4499785)  
GPT-5.4 모델은 복잡한 멀티스텝 워크플로우를 실행할 수 있는 신뢰성과 고급 추론 기능을 제공합니다. 실시간 처리와 높은 품질의 결과물을 제공합니다.

### [VibeVoice ASR 정식 지원](https://techcommunity.microsoft.com/t5/microsoft-foundry-blog/introducing-vibevoice-asr-longform-structured-speech-recognition/ba-p/4501276)  
장시간 오디오를 단일 패스로 변환하는 VibeVoice ASR은 음성 인식 모델로, 회의 녹음과 팟캐스트 등 장문의 오디오 데이터 구조화에 기여합니다.

### [Fireworks AI 미리보기 출시](https://techcommunity.microsoft.com/t5/microsoft-foundry-blog/announcing-fireworks-ai-on-microsoft-foundry/ba-p/4500950)  
Fireworks AI는 오픈 소스 모델을 빠르고 효율적으로 배포할 수 있도록 설계된 고성능 AI 서비스입니다.

## 🧩 Microsoft Fabric

Microsoft Fabric 업데이트는 데이터 저장소와 분석을 최적화하고, 데이터 관리 효율성을 높이는 데 중점을 두었습니다. OneLake Storage 및 데이터 유지 정책이 핵심입니다.

### [OneLake 저장소 라이프사이클 관리 정책 미리보기](https://roadmap.fabric.microsoft.com/?product=onelake)  
자동 데이터 전환 정책을 적용해 저장 비용 최적화를 지원합니다. 데이터 액세스 또는 수정 시간을 기준으로 핫, 쿨, 콜드 티어 간의 이동을 자동화합니다.

### [구성 가능한 데이터 유지 정책 미리보기](https://roadmap.fabric.microsoft.com/?product=datawarehouse)  
데이터웨어하우스 관리자가 데이터 보존 기간을 1~120일로 설정할 수 있어 더욱 유연한 데이터 관리가 가능합니다.

### [Fabric Activator, Ontology Rules 통합](https://blog.fabric.microsoft.com/en-US/blog/from-insight-to-action-bringing-fabric-activator-into-ontology-with-rules/)  
Fabric Activator와 Ontology Rules는 실시간 데이터 기반 의사결정을 더욱 직관적이고 자동화된 방식으로 지원합니다.

보안을 강화한 전용 네트워크를 사용해 간단하면서도 안전한 데이터 전송이 가능해졌습니다.
