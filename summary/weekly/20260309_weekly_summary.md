# Azure 주간 업데이트 요약 - 2026년 03월 09일

## 💡 정식 지원 (General Availability)

지난주 Azure는 여러 서비스에서 운영 준비가 완료된 정식 지원 기능을 공개했습니다.

### [Introducing GPT-5.4 in Microsoft Foundry](https://techcommunity.microsoft.com/t5/microsoft-foundry-blog/introducing-gpt-5-4-in-microsoft-foundry/ba-p/4499785)
GPT-5.4 모델은 다단계 워크플로우와 장기적인 신뢰성을 지원하며, 통합된 AI 작업 제공을 위한 다양한 개선 사항을 포함합니다.

### [Azure Databricks Lakebase](https://azure.microsoft.com/updates?id=557991)
Azure Databricks Lakebase는 실시간 분석과 AI 기반 애플리케이션을 제공하며, 데이터 이동을 줄이고 복잡한 ETL 프로세스를 제거합니다.

### [Draft & Deploy on Azure Firewall](https://azure.microsoft.com/updates?id=558072)
Azure Firewall에 다중 수정 기능이 추가되어 정책 수정과 배포가 더욱 효율적으로 진행됩니다.

## 🚀 미리 보기 (Preview)

지난주에는 새롭게 시험 가능한 Azure 기술들의 미리 보기 업데이트가 발표되었습니다.

### [Fabric SQL DB in Workspace Monitoring](https://roadmap.fabric.microsoft.com/?product=sqldatabase)
SQL DB의 워크스페이스 모니터링이 Fabric 내에 통합되었으며, 2026년 6월 30일 출시 예정입니다.

### [Rules for Ontology](https://roadmap.fabric.microsoft.com/?product=iq)
Ontology 규칙은 비즈니스 프로세스 자동화를 지원하며, 실시간 모니터링 및 알림 기능을 통합합니다.

### [Eventstream Connectors in Microsoft Fabric](https://blog.fabric.microsoft.com/en-US/blog/powering-secure-private-network-streaming-to-fabric-with-eventstream-connectors-preview/)
Fabric에서 안전한 데이터 스트리밍을 지원하는 새로운 Eventstream Connectors가 미리 보기로 제공됩니다.

### [Grok 4.0 & Grok 4.1 Fast 출시](https://techcommunity.microsoft.com/t5/microsoft-foundry-blog/grok-4-0-goes-ga-in-microsoft-foundry-and-grok-4-1-fast-arrives/ba-p/4497964)

### [Microsoft Agent Framework RC](https://devblogs.microsoft.com/foundry/microsoft-agent-framework-reaches-release-candidate/)
