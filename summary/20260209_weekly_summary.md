# Azure 주간 업데이트 요약 - 2026-02-09

## 🚀 정식 지원된 기능 (Launched)

Azure는 사용자를 위한 새로운 정식 지원 제품 및 기능을 지속적으로 제공하여 플랫폼의 성능과 기능을 향상시키고 있습니다. 이번 주에는 보안, 컨테이너, 애널리틱스와 관련된 주요 Azure 업데이트가 발표되었습니다. 이를 통해 사용자는 더욱 빠르고 효율적인 작업 수행이 가능합니다.

### 🌐 [Default Ruleset 2.2 in WAF for Azure Application Gateway](https://azure.microsoft.com/updates?id=553532)
Azure WAF(Web Application Firewall)의 Default Rule Set (DRS) 2.2이 정식 지원으로 출시되었습니다. 이 규칙 세트는 웹 어플리케이션의 일반적인 취약성을 보호하며, Microsoft Threat Intelligence와 연계된 보호 기능을 제공합니다.

### 🖥️ [Serverless Workspaces in Azure Databricks](https://azure.microsoft.com/updates?id=550845)
Azure Databricks의 서버리스 워크스페이스 기능이 정식 지원되었습니다. 이 기능은 사용자가 최소 설정으로 워크스페이스를 관리하며 빠르게 생산 환경을 구축하고 테스트할 수 있도록 돕습니다.

### ⚙️ [AMD v6 Confidential VMs in New Regions](https://azure.microsoft.com/updates?id=553966)
AMD 기반 v6 컨피덴셜 가상머신이 11개의 새로운 지역에서 지금 사용할 수 있습니다. 이 업데이트는 글로벌 확장을 지원하며 다양한 지역에서 데이터 보안 인프라를 제공합니다.

## 🔍 미리 보기 기능 (In Preview)

Azure의 미리 보기 기능은 혁신적인 기술과 안정성을 탐구하는 사용자에게 제공됩니다. 미리 보기 단계는 새 설계와 기능에 대한 사용자 피드백을 반영하며 서비스 품질 개선에 초점을 둡니다.

### 🔗 [Azure Kubernetes Fleet Manager Namespace-Scoped Resource Placement](https://azure.microsoft.com/updates?id=548198)
Azure Kubernetes Fleet Manager에서 네임스페이스별 리소스 배치 기능이 추가되었습니다. 이를 통해 다양한 클러스터 내 복잡한 애플리케이션 워크로드를 간편하게 관리할 수 있습니다.

### 🧠 [Claude Opus 4.6 Available on Microsoft Foundry](https://azure.microsoft.com/updates?id=555870)
Claude Opus 4.6이 Microsoft Foundry 플랫폼에서 사용할 수 있게 되었습니다. 이 모델은 인공지능의 고급 추론 및 코드 작업을 지원하여 사용자가 혁신적인 업무를 처리할 수 있도록 돕습니다.

### 🛠️ [Azure Monitor Pipeline Data Transformations](https://azure.microsoft.com/updates?id=555488)
Azure Monitor 파이프라인 데이터 변환이 베타 기능으로 출시되었습니다. 이 업데이트는 데이터 품질을 향상시키고 분석을 간소화하여 복잡한 텔레메트리 환경에서 정보를 쉽게 처리할 수 있도록 설계되었습니다.

## ⚙️ 개발 중인 기능 (In Development)

Azure는 지속적으로 새로운 기술을 개발하며 글로벌 클라우드 혁신을 선도합니다. 이번 주에는 새로운 데이터 센터와 향후 출시될 주요 기능에 대한 계획이 공개되었습니다.

### 📊 [Public Preview: X-Forwarded-For Grouping for Rate Limiting](https://azure.microsoft.com/updates?id=555205)
X-Forwarded-For 헤더 기반 속도 제한 기능이 Application Gateway WAF v2에 추가될 예정입니다. 이를 통해 보안 팀은 원래 사용자 IP를 기반으로 보다 정확한 속도 제한을 설정할 수 있습니다.

### 🌏 [New Planned Data Center Region in Thailand](https://azure.microsoft.com/updates?id=553999)
태국 남부 지역에 새로운 데이터 센터 설립 계획이 발표되었습니다. 이 지역은 Azure의 확장된 클라우드 서비스를 제공하며 데이터 보안과 성능을 강화할 것입니다.

### 💾 [Vaulted Backups for Azure Disk](https://azure.microsoft.com/updates?id=555184)
Azure Disk Backup에 새로운 Vault 기반 백업 기능이 추가될 예정입니다. 이 기능은 랜섬웨어 탄력성과 보안을 강화하여 데이터 복구를 지원합니다.

## 💻 네트워킹 및 보안 업데이트

Azure의 네트워킹 및 보안 분야 업데이트는 데이터와 애플리케이션을 안전하게 보호하며 최적의 연결성을 제공합니다. 이번 주에는 중요한 보안 관련 업데이트가 포함되었습니다.

### 🛡️ [Azure Front Door and CDN Cipher Suites Retirement](https://azure.microsoft.com/updates?id=553527)
Azure Front Door 및 CDN의 약한 암호화 수트 지원 종료가 발표되었습니다. 이는 보안 강화를 위한 조치로, 사용자에게 암호화 업그레이드를 권장합니다.

### 🚦 [Azure Virtual Network Routing Appliance](https://azure.microsoft.com/updates?id=555944)
Azure 가상 네트워크 라우팅 어플라이언스가 미리 보기 단계로 출시되었습니다. 이를 통해 향상된 성능과 낮은 대기 시간을 가진 네트워크 연결을 활용할 수 있습니다.

### 🌐 [Default Ruleset 2.2 in WAF for Azure Application Gateway](https://azure.microsoft.com/updates?id=553532)
Azure WAF의 DRS 2.2가 네트워크 보안을 강화하기 위한 필수 보안 업데이트로 출시되었습니다.

### 🛠️ [Azure Monitor Pipeline Data Transformations](https://azure.microsoft.com/updates?id=555488)
Azure Monitor에서 실시간 데이터 변환 기능이 제공되며 데이터 품질과 분석 속도를 크게 개선합니다.
