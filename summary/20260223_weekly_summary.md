# Azure 주간 업데이트 요약 - 2026-02-23

## 🛠️ 신규 기능 및 출시 (Launched Features)
Azure는 최근 혁신적인 기능들을 출시하며 사용자들에게 향상된 클라우드 경험을 제공하고 있습니다. Azure Premium SSD v2 및 Ultra Disk의 즉시 액세스 스냅샷 지원은 백업 및 복구 워크플로를 신속하게 처리할 수 있도록 하며, Azure Functions에서 .NET 10에 대한 정식 지원은 서버리스 애플리케이션에 최신 플랫폼 기능을 적용할 수 있게 합니다. 또한, AKS(Azure Kubernetes Service)의 노드 자동 프로비저닝 및 로컬 DNS 지원은 클라우드 리소스 관리의 효율성을 극대화합니다.

### [Azure Premium SSD v2 및 Ultra Disk 즉시 액세스 지원](https://azure.microsoft.com/updates?id=545784)
Premium SSD v2와 Ultra Disk의 스냅샷 복구를 즉시 처리할 수 있는 기능이 업데이트되어 환경 복구와 규모 확대가 더 신속하게 이루어질 수 있습니다.

### [Azure Kubernetes Service 노드 자동 프로비저닝에서 호스트 및 디스크 암호화 지원](https://azure.microsoft.com/updates?id=557213)
노드 자동 프로비저닝 클러스터에 암호화 기능이 추가되어 보안 요구 사항을 충족하면서 리소스 효율성을 극대화할 수 있습니다.

### [Azure Functions의 .NET 10 정식 지원](https://azure.microsoft.com/updates?id=556003)
최신 .NET 런타임을 서버리스 애플리케이션에 적용할 수 있으며, 새로운 기능과 성능 향상을 제공합니다.

---

## 📦 컴퓨팅 및 컨테이너 기능 (Compute & Containers)
Azure의 AKS는 클라우드 기반 애플리케이션 관리를 강화하기 위해 다양한 업데이트를 도입했습니다. 노드 자동 프로비저닝을 통한 로컬 DNS 지원과 정부 및 프라이빗 클라우드 환경에서의 자동화된 컴퓨팅 관리가 주요 진전입니다. 또한 클러스터 모드 관리 도구의 미리 보기를 통해 보안 및 운영 효율성을 향상시킬 수 있습니다.

### [AKS 노드 자동 프로비저닝에서 LocalDNS 지원](https://azure.microsoft.com/updates?id=557203)
노드 자동 프로비저닝이 활성화된 클러스터에 LocalDNS 사용이 가능해져 클라우드 환경 내부에서의 DNS 관리를 더욱 최적화할 수 있습니다.

### [Azure 정부 및 프라이빗 클라우드에서 Node Auto-Provisioning 지원](https://azure.microsoft.com/updates?id=557208)
정부 및 프라이빗 클라우드 환경에서도 자동 리소스 관리 기능이 제공되며, 작업 부하 요구 사항에 따라 프로비저닝과 관리를 자동화합니다.

### [AKS를 위한 에이전트 클러스터 모드 미리 보기](https://azure.microsoft.com/updates?id=557218)
클러스터 모드에서 보안 제어와 진단을 강화하고, Kubernetes 및 Azure 데이터를 통합적으로 관리하는 기능을 제공합니다.

## 💾 저장소 및 데이터 관리 (Storage & Data Management)
Azure는 고성능 스토리지를 확장하며 데이터 관리 솔루션을 강화했습니다. 특히 Azure Premium SSD v2가 브라질, 말레이시아, 인도네시아 등 더 많은 글로벌 지역에서 제공됨으로써 클라우드 스토리지의 접근성과 성능을 확대했습니다. 이로 인해 데이터 처리와 분석 작업이 대폭 향상되었습니다.

### [Azure Premium SSD v2 지역 확장](https://azure.microsoft.com/updates?id=557623)
브라질, 말레이시아, 인도네시아 지역에서 Azure Premium SSD v2를 사용할 수 있게 되어 글로벌 데이터 관리 시스템을 더욱 강화합니다.

## ⚙️ AI와 머신러닝 (AI & Machine Learning)
Azure Databricks는 새로운 AI 모델 Anthropic Claude Sonnet 4.6을 추가하여 복잡한 코딩 영역에서 더욱 혁신적인 해법을 제공합니다. 이 모델은 고급 분석 및 연구 워크로드를 위해 설계되었으며, 사용자에게 더 빠르고 신뢰할 수 있는 AI 환경을 제공하고 있습니다.

### [Azure Databricks에서 Anthropic Claude Sonnet 4.6 지원](https://azure.microsoft.com/updates?id=557595)
새로운 AI 모델은 코딩 작업 및 AI 에이전트 빌드에 최적화된 성능을 제공하며 Azure Databricks 환경에서 사용할 수 있습니다.

Azure Functions의 Flex Consumption 진단 및 문제 해결 도구는 운영 인사이트를 크게 개선하며, 클라우드 작업 환경에서 발생하는 비효율성을 줄일 수 있도록 지원합니다. 이는 새로운 운영 툴과 함께 제공됩니다.

### [Azure Functions Flex Consumption 문제 해결 도구](https://azure.microsoft.com/updates?id=556008)
새로운 툴은 Flex Consumption 계획에 대한 구축 및 작업 문제를 효율적으로 진단하고 해결할 수 있도록 지원합니다.
