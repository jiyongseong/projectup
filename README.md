# 🚀 Azure Service Updates Weekly Summary

> **"매일 쏟아지는 Azure의 방대한 업데이트, 이제 주 단위 요약으로 핵심만 빠르게 파악하세요!"**

이 저장소는 Microsoft Azure의 [공식 업데이트 페이지](https://azure.microsoft.com/ko-kr/updates)에 게시되는 매일의 기술 소식들을 수집하여, 클라우드 엔지니어와 아키텍트가 읽기 좋게 **주 단위(Weekly)** 로 요약하여 제공합니다.

해당 저장소의 내용은 **Project UP(date) by SE태지**에 의해서 자동으로 생성됩니다.

---

## 📌 프로젝트의 목적 

Azure 생태계는 매우 빠르게 변화합니다. 매일 업데이트를 확인하기 어려운 바쁜 전문가들을 위해 다음과 같은 가치를 제공합니다.

* **핵심 요약:** 수많은 공지 중 기술적 우선순위가 높은 내용을 선별합니다.
* **카테고리 분류:** GA(General Availability), Preview, Retirement 소식을 체계적으로 분류합니다.
* **히스토리 아카이브:** 특정 시점에 어떤 기능이 추가되었는지 쉽게 검색하고 추적할 수 있습니다.

---

## 📂 저장소 구조

업데이트 리포트는 다음의 폴더에 정리되어 있습니다.

- [**pdfs 폴더**](./pdfs/)에는 주간 업데이트 전체 내용이 포함된 pdf 파일들이 있으며
- [**summary 폴더**](./summary/)에는 주간 업데이트 요약 markdown 문서가 있습니다.

---

## 🛠️ 활용 방법

1. **Issue/Discussions:** 특정 업데이트에 대한 심도 있는 논의가 필요하면 Issue를 활용해 주세요.
2. **Watch:** 이 저장소를 **Watch** 설정하면 매주 업데이트되는 Azure 소식을 알림으로 받아보실 수 있습니다.
3. **Search:** `Ctrl + F`를 통해 특정 서비스(예: AKS, SQL, Sentinel)의 최신 변경 사항을 찾으세요.

## 🤝 기여하기

이 프로젝트는 커뮤니티의 참여를 언제나 환영합니다!
